# 설명
KBO경기의 경우 홈팀과 원정팀의 선발선수들의 최근 WAR성적을 입력한다.              
MLB경기의 경우 각 팀의 평균 ERA  WHIP  투수WAR SLG OBP 타자WAR을 입력한다.      
NBA경기의 경우 각 팀의 평균 WS VORP BPM을 입력한다.
승리팀이 HOME인지 AWAY인지를 JSON으로 return해준다.
